Заголовок и описание проекта:
Проект называется "web-application". В этом веб-приложение использует обученную модель нейронной сети для предсказания цены автомобиля с помощью нескольких признаков(также есть такой доп.контент элементы как картинки и видео)

Установка:
1. Загрузить себе проект на локальный компьютер
2. Запустить в Visual Studio Code
3. Открыть в Visual Studio Code папку с проектом и открыть файл "web-application.py"
4. В терминале выполнить команду "pip install streamlit"
5. В терминале выполнить команду "streamlit run web_application.py"
6. Используем наше веб-приложение!

Использование:
Запускаем видео, выбираем значения в соответсвующие поля, нажимаем на кнопку и смотрим результат!
